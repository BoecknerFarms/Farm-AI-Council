# Farm AI Council

Local multi-agent farm decision system.

Initial agents:
- Managing Agent
- Agronomist Agent
- Historian Agent

Purpose:
The system allows specialist AI agents to discuss farm scenarios, review prior farm history, and produce a practical management recommendation.

Initial version:
- Runs locally through Ollama
- Uses local agent prompt files
- Stores conversations and reports for future memory
- Does not yet use live web data or full document retrieval
